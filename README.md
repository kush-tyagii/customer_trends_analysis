# customer_trends_analysis
Customer Behavior Analysis is an end-to-end data analytics project that analyzes customer purchasing patterns, demographics, and spending behavior using Python, SQL, and data visualization techniques.

## 📌 Project Overview

This project analyzes customer shopping data to answer business-related questions such as:

- Which customer segments spend the most?
- What are the most popular product categories?
- Which payment methods are preferred?
- How does shipping type affect purchasing behavior?
- What trends can be observed across age groups and genders?

The project combines **Python for data analysis** and **SQL for business queries**, making it suitable for aspiring Data Analysts and Data Engineers.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Analyze customer purchasing patterns
- Write SQL queries for business insights
- Create meaningful visualizations
- Generate actionable business recommendations
